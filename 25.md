例1-11。切片和索引张量

```python
import torch

```

```

 describe(x[:1, :2])
 ```
 - output 1
 ```
 Type: torch.FloatTensor

tensor([[ 0.,  1.]])
 ```
 - input 2
 ```python
 describe(x[0, 1])
 ```
 - output 2
 ```
 Type: torch.FloatTensor
Shape/size: torch.Size([])
Values:
1.0
 ```
