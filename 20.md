我们也可以创建张量都充满了同样的标量。对于创建的0或1张量，我们
有内置函数，以及用于与特定值填充它，我们可以使用fill_（）方法。 任何
PyTorch方法用下划线（_）指的是就地操作;即，它修改
代替内容而不创建新对象，如图
实施例15。
- input
```python
import torch
describe(torch.zeros(2, 3))
x = torch.ones(2, 3)
describe(x)
x.fill_(5)
describe(x)
```
